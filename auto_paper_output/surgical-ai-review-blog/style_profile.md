# Style Profile

## Target Voice

中文深度技术博客。语气应该是清晰、判断明确、但不过度学术化。文章可以有作者观点，但每个强判断都要带边界。

## Style Rules

- Use direct section titles rather than paper-like subsection numbering.
- Prefer `趋势判断`, `可能路线`, `值得关注` over `已经证明`, `必然`, `终局`.
- Explain technical terms with surgical meaning: 3D reconstruction is not the same as navigation, visible surface is not the same as deep anatomy.
- Mention studies as `材料中提到的研究` unless independently verified.
- Keep the final blog readable without requiring the source PDF.
