# Motivation Thread

## Confirmed Motivation Fields

`confirmed_motivation.md` must include:

- `one_sentence_argument`
- `field_need`
- `unresolved_bottleneck`
- `proposed_move`
- `decisive_evidence`
- `broader_implication`
- `boundary`
- `claims_to_avoid`

## Surface Map

Map the red thread to:

- title
- abstract
- introduction final paragraph
- method opening
- experiment setup
- discussion first paragraph
- limitation paragraph

## Claim Strength

Use allowed verbs that match evidence strength. Reserve novelty, SOTA,
clinical, generalization, and statistical language for claims that have
explicit author evidence and citation support.

## Common Chinese-Author Workflow

It is acceptable to reason about motivation in Chinese, write claims in
English, audit risks in Chinese, and patch the final manuscript in the target
venue language. Do not translate Chinese logic into long literal English
sentences; rewrite the claim function for the target reader.
