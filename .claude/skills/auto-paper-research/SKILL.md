# Auto Paper Research

<instructions>
1. Read intake artifacts before any synthesis.
2. Summarize author evidence, known claims, gaps, and missing files with
   provenance.
3. Learn exemplar structure and style without creating author claims.
4. Map SOTA gaps from local bib/notes and optional user-requested search.
5. Write `research_dossier.md`, `exemplar_learning_dossier.md`,
   `style_profile.md`, and `sota_gap_map.md`.
</instructions>
