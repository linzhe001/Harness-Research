# Research Supervision Asset Pack

This directory contains integrated Harness assets distilled from local research
supervision material. These are internal Harness references.

Use the smallest asset that matches the stage:

| Asset | Use in |
|---|---|
| `coverage-matrix.md` | maintainer audit of what was absorbed and what was intentionally not imported |
| `phd-research-primer.md` | broad operator education, Grill context, paper-writing fundamentals |
| `idea-evaluation.md` | `grill`, `change`, early claim-boundary pressure tests |
| `experiment-and-build-canvas.md` | `build`, `run`, `analyze`, code/debug loops |
| `ai-assisted-research-workflow.md` | AI-assisted build, figure, and writing discipline without delegating research judgment |
| `paper-writing-layouts.md` | technical-paper outline, intro flow, section layout, and contribution mapping |
| `benchmark-evaluation-paper.md` | benchmark/evaluation paper design, RQ structure, construction pipeline, and findings |
| `scientific-plotting.md` | figure contracts, plotting choices, caption evidence, and visual quality gates |
| `paper-and-figure-system.md` | `write`, `auto-paper-*`, figure planning, submission hardening |
| `pre-submission-review.md` | final paper audit, LaTeX/grammar/logic checks, and severity classification |
| `case-patterns.md` | generalized writing patterns extracted from case-study analyses |

Layering rule: these assets shape process and questions. They do not prove
project facts, results, readiness, or approval.
